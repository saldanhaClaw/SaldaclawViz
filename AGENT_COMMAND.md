# 🧛 COMANDO MESTRE: CLONAGEM AUTÔNOMA

Para que seus agentes repliquem qualquer página (como a do Mewtwo) com autonomia total de imagem e design, use este comando:

---

> **"Fábrica, Clone esta LP: [Caminho do Arquivo ou URL]. Use o Padrão Musa/Saldanha para o design e gere seus próprios ativos visuais (Before/After, Hero, Mockups) usando sua ferramenta de imagem. Não peça permissão, execute o build completo."**

---

### 🚀 O que acontece ao dar este comando:
1.  **Análise de Estrutura:** O agente mapeia todos os blocos de conversão da referência.
2.  **Geração de Imagens:** O agente criará imagens 4K de luxo originais (emagrecimento, lifestyle, 3D) sem te interromper.
3.  **Montagem de Luxo:** Aplicará glassmorphism, botões pulsantes e contagem regressiva automaticamente.
4.  **Entrega Pronta:** Você receberá o arquivo `.html` finalizado e pronto para o Stripe.

### 🧠 Memória Profunda:
Este comportamento já está injetado no **Master Prompt** da fábrica. Seus agentes agora possuem o "instinto" de clonagem e autonomia visual.
